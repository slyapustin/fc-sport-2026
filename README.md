# FC Sport 2026

A 3D football game built with Three.js — playable entirely in the browser.

## Play

[https://slyapustin.com/fc-sport-2026/](https://slyapustin.com/fc-sport-2026/)

## Features

- 3D match gameplay with Three.js
- Multiple leagues and teams to choose from
- Touch and keyboard controls
- Single-file, zero-build — just open `index.html`
